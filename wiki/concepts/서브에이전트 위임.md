# 서브에이전트 위임 (Subagent Delegation)

탐색·조사 작업을 별도 프로세스(서브에이전트)에 위임하여 메인 컨텍스트 창을 보호하는 기법.

## 원리

메인 컨텍스트에서 10개 파일을 탐색하면 모든 파일 내용이 컨텍스트에 누적된다.
서브에이전트에 위임하면 요약 결과만 메인 창으로 반환된다.

## 사용 기준

| 작업 | 위임 여부 |
|---|---|
| 1~2개 파일 단순 읽기 | 직접 실행 (오버헤드가 더 큼) |
| 3개 이상 파일 탐색 | 서브에이전트 위임 |
| 파일 간 종속성 조사 | 서브에이전트 위임 |
| 테스트 실행 및 결과 요약 | 서브에이전트 위임 |
| 코드 리뷰 (파일별 병렬) | 서브에이전트 위임 |

## CLAUDE.md 규칙 예시

```markdown
# Cost Optimization
- Delegate exploration of 3+ files to subagents
- Run tests via subagents and return results only
- For code reviews, launch parallel subagents per file
```
