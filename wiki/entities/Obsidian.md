# Obsidian

마크다운 기반 개인 지식 관리(PKM) 도구.

## 이 볼트에서의 역할

LLM이 wiki 파일을 작성하면 Obsidian에서 실시간으로 탐색한다.
**LLM은 프로그래머, Obsidian은 IDE, wiki는 코드베이스.**

## 주요 기능 (이 볼트 관련)

- **Graph View** — wiki 페이지 간 링크를 지식 그래프로 시각화
- **Dataview** — YAML 프론트매터 기반 동적 쿼리
- **Web Clipper** — 웹 아티클을 마크다운으로 변환해 `raw/`에 저장
- **Canvas** — graphify가 생성한 커뮤니티 레이아웃 시각화
